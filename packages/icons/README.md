# @m3e/icons

The @m3e/icons package provides SVG‑based icon modules for M3E Web Components, supplying icons for use with the `m3e-icon` component. Each icon is a standalone ES module that registers itself when imported.

## Examples

### Import a single icon

```js
import "@m3e/icons/outlined/search";
```

### Import all icons for a variant

```js
import "@m3e/icons/outlined";
import "@m3e/icons/rounded";
import "@m3e/icons/sharp";
```

### Import all icons

```js
import "@m3e/icons";
```
