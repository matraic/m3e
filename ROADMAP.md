# 📍 M3E Roadmap

This roadmap presents the ongoing development, future direction, and past accomplishments of M3E.

## 🟡 Current

- 2.1.0 Release

## 🔮 Future

### ✨ Components

The following components are planned for future release.

#### 🛠️ In Progress

Components in nearing completion include:

- Calendar
- Datepicker

Components in early stages of development include:

- Skeleton Loader
- Split Pane
- Tree

#### 🔮 Not Yet Started

Planned components which have not yet entered development:

- Inputs (Numeric, Masked, Date, Datetime)
- Time Picker
- Search

### 📜 Building with AI

- Global and component-specific `llms.txt` files to support AI discoverability
- MCP Server enabling IDEs and AI tools to query local knowledge about components and documentation

## 🏛️ Past Milestones

- **2.1.0 – 2.1.2 — Feature + patch releases**
  Enhancements to `m3e-autocomplete` and `m3e-input-chip-set`, bug fixes.

- **2.0.1 – 2.0.5 — Patch releases**  
  Bug fixes, SSR support for React.

- **2.0.0 — Major release**  
  Unified architecture under `@m3e/web`, introduced `@m3e/react` (React bindings), consolidated components, and deprecated legacy per‑component packages.

- **1.3.0 – 1.3.1 — Feature + patch releases**  
  Added `@m3e/bottom-sheet` and delivered follow‑up bug fixes.

- **1.2.1 – 1.2.2 — Patch releases**  
  Bug fixes.

- **1.2.0 — Feature release**  
  Bug fixes, expressive updates to `@m3e/menu`, and introduction of the rich tooltip in `@m3e/tooltip`.

- **1.1.8 – 1.1.11 — Patch releases**  
  Bug fixes.

- **1.1.7 — Feature release**  
  Added `@m3e/avatar`, expressive updates to `@m3e/list`, and new selection/action patterns.

- **1.1.4 – 1.1.6 — Patch releases**  
  Bug fixes.

- **1.1.3 — Feature release**  
  Bug fixes and added the wavy appearance variant to `@m3e/progress-indicator`.

- **1.1.1 – 1.1.2 — Patch releases**  
  Bug fixes.

- **1.1.0 — Feature release**  
  Added `@m3e/paginator`, bundle size reductions, and RTL layout fixes.

- **1.0.6 — Feature release**  
  Added `@m3e/autocomplete`, `@m3e/select`, and `@m3e/option`.

- **1.0.5 — Patch release**  
  Bug fixes and converted `@m3e/all` into a pure meta‑package.

- **1.0.4 — Patch release**  
  Bug fixes.

- **1.0.3 — Feature release**  
  Added `@m3e/stepper`.

- **1.0.1 – 1.0.2 — Patch releases**  
  Bug fixes.

- **1.0.0 — Stable release**  
  First full, stable version of the M3E component library.

- **1.0.0 RC1 – RC4 — Release candidates**  
  Initial component development, testing, and documentation.
